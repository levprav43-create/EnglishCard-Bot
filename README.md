# EnglishCard — Telegram-бот для изучения английского

Курсовая работа по курсу «Базы данных» в Нетологии.

## Возможности
- Перевод слов (русский ↔ английский)
- Тесты с 4 вариантами ответа
- Добавление и удаление своих слов
- Регистронезависимый поиск
- Счётчик количества слов у пользователя

## Технологии
- Python 3 + pyTelegramBotAPI
- PostgreSQL + SQLAlchemy (ORM)
- Git + GitHub

## Запуск
1. Создайте `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
